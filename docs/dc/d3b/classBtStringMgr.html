Detailed Description

Unicode string manager implementation.

A Qt-based sword::StringMgr is better than the default one in Sword, in case Sword is not compiled against ICU regarding this. However, we have no good means to check this at compile time, hence we provide this class. It is initialized in BibleTime::initBackends() as follows:

if (!sword::SWMgr::isICU)
sword::StringMgr::setSystemStringMgr(new BtStringMgr());
