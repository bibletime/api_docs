67 /**
68  * @page modelviewmodel Details about the module text model
69  * <p>
70  * The BtModuleTextModel represents the text of a sword module. It uses
71  * an integer index to address each entry (verse, paragraph, etc).
72  * The html text is obtained using backend rendering functions.
73  * It is then post processed to fix a few things so that the QML
74  * ListView works correctly. The ListView supports a "rich text"
75  * that is a subset of html. The post processing is done in
76  * the class BtTextFilter which is installed into BtModuleTextModel
77  * by the frontend.
78  * </p>
79  */
80

83  \brief Model that represents the entire text of a given module

85  It is used by a QML ListView to view a small portion of the
86  available module text. Only the portions of the module that are

88  instantiated.
